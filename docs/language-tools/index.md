# Language Tools

## What This Section Contains

This section is the reusable language toolbox of the course.

Unlike the modules, which follow a teaching sequence, the pages here are designed for quick consultation while reading, drafting, revising, or preparing technical communication.

Use this section when you need support with:
- grammar for technical writing;
- functional language for specific communication tasks;
- linking words and text organisation;
- phrasebanks for common engineering contexts;
- common mistakes and false friends.

## At a Glance

The Language Tools section includes three main resource types:

| Resource Type | Purpose |
|---|---|
| Core references | explain grammar, function, and text organisation |
| Phrasebanks | provide reusable wording for specific tasks |
| Error-prevention pages | help avoid recurring mistakes and transfer problems |

## When to Use This Section

Use these pages at different stages of the writing process.

### Before Drafting
Consult the relevant phrasebank or functional-language page to choose the right tone and structure.

### During Drafting
Use the grammar and linking pages to keep the text clear, controlled, and logically connected.

### During Revision
Use the common-mistakes and false-friends pages to detect weak phrasing, interference, or inappropriate wording.

## Practical Advice

Do not try to memorise entire pages.

A better approach is to:
1. identify the communication task;
2. select the relevant expressions;
3. use them in a draft;
4. revise the draft for clarity and consistency.

## Final Note

This section is not a grammar appendix.

It is a working resource designed to help the learner produce technical English that is clearer, more controlled, and more reusable in professional documentation.
