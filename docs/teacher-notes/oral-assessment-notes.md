# Oral Assessment Notes

!!! note "Teaching use note"
    These notes are intended to stabilise correction behaviour across tasks, modules, and repository outputs. Use them to keep assessment principles explicit and repeatable.

## What matters most in technical speaking

Technical speaking should be judged primarily on:

- sequencing;
- terminology;
- clarity of logic;
- evidence awareness;
- control of scope.

The student does not need to sound theatrical. A calm, structured, engineering-like briefing is preferable to an energetic but vague presentation.

## Correction points

- Did the learner distinguish fact from working hypothesis?
- Did the learner manage transitions clearly?
- Were key terms repeated consistently?
- Was the closing statement useful and action-oriented?
