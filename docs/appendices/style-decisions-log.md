# Style Decisions Log

## Purpose

This appendix records editorial choices that affect the repository as a whole. It is especially useful when multiple contributors are maintaining the same documentation environment.

## Current decisions

| Area | Decision | Rationale |
| --- | --- | --- |
| Register | Neutral professional English | Supports civil and defence-oriented contexts without tone drift |
| Glossary use | Repetition allowed where it improves precision | Technical documentation values stability over stylistic variety |
| Admonitions | Used for purpose, caution, review notes, and editorial guidance | Prevents decorative overuse |
| Templates | Structured but not over-scripted | Encourages adaptation rather than mechanical copying |
| Cross-links | Functional links preferred over exhaustive linking | Keeps navigation meaningful |

## Update rule

When a new style decision changes repository behaviour in a lasting way, record it here and align the relevant playbooks, teacher notes, or templates accordingly.
