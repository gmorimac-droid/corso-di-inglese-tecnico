# Appendices

!!! abstract "Section purpose"
    The appendices gather repository-wide conventions that support control, readability, and long-term maintenance.

## Contents

- approved abbreviations and acronyms;
- document-control conventions;
- style decisions log.

## Why appendices matter

A documentation repository becomes fragile when basic conventions remain implicit. Appendices help stabilise decisions that would otherwise be reinterpreted inconsistently by different contributors.
