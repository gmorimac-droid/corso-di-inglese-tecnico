# Writing Quality Playbook

!!! note "Playbook use note"
    A playbook is most useful when applied in sequence. Follow the review order rather than correcting isolated sentences without a framework.

## Premium writing checks

A premium engineering-English page should normally satisfy the following criteria:

- the first paragraph defines scope and purpose;
- terminology remains stable;
- sentence length is controlled;
- lists are used where they improve skimmability;
- uncertainty is marked honestly;
- conclusions are narrower than evidence, not wider.
